# Zelda-Project
Adaptation from Zelda "A link to the past" game made for our Videogames Degree

=================================================================
CONTROLS

CHANGING SCENE:	--> THIS IS A NON-DEFINITIVE KEY FOR THIS RELEASE
-F4

Movement:
- WASD
Attack:
- Arrow Keys
Change Weapon:
- Q/E
PickUp:
- Space Bar
Dash:
- Right Control

UI:
Inventory:
- TAB
Menu:
-ESC
Debug Mode:
- F1
Free Camera Movement: Arrow Keys
- F2
Collider Debug
In Main- Menu:
-UP-DOWN:move selector
-ESC: back
Change Controls:
-press the desired control, when pressed, press the key you want to assign.
